How To Install the Mods 

Download the document from the above links.
Open the Arma 3 Launcher and select the Mods Tab for MODS tab (1)
Select the PRESET drop down menu at the top right corner (2)
At the bottom of the preset tab click import (3) and select the HTML file downloaded earlier (will be in downloads folder on PC).
Load the pack and it will automatically Subscribe the workshop content for you. 
Ensure the mods are loaded before pressing Play

The teamspeak plugin is required to run alongside our modpack, it enables teamspeak to integrate communication with Arma 3, download it and run it, this will install it into TS3.

Once installed open teamspeak and open the Tools menu and select Options, then select the Addons section and ensure that Task Force Arrowhead Radio is showing it is Enabled.

Once you login to the arma 3 server and spawn in, it will automatically drag you into our Task Force Radio channel in teamspeak allowing you to communicate with other players via teamspeak with a directional and distance variable VOIP.
